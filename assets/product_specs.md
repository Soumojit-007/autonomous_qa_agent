# Product Specifications

- The checkout page allows users to enter their name, email, and address.
- The "Pay Now" button submits the form and triggers payment processing.
- Users must select a shipping method: "standard" (free) or "express" (paid).
- Users must choose a payment method: "Credit Card" or "PayPal".
- Discount code input is available, but discount logic is applied client-side.
- The discount code "SAVE15" applies a 15% discount on total amount.
